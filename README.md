# Modern-Translator-CustomTkinter

Легковесный десктопный переводчик текстов с поддержкой автоматического определения исходного языка. Интерфейс приложения построен на базе библиотеки customtkinter, обеспечивающей современный внешний вид элементов управления и нативную поддержку темной/светлой тем операционной системы. За логику перевода отвечает стабильный движок deep-translator (Google Translate API под капотом).

Проект разработан в качестве практического примера построения клиентских графических интерфейсов на Python.
## ✨ Ключевые возможности

    *Асинхронное автоопределение: Корректная обработка параметра auto. Приложение переводит текст и параллельно информирует пользователя о том, какой язык был обнаружен в текстовом буфере.

    *Реверс направлений перевода (Swap): Фича быстрой смены целевого и исходного языков кнопкой ⇄ с автоматической валидацией граничных условий (блокировка установки «Автоопределения» в качестве конечного языка).

    *Современный UX: Динамическое изменение состояния текстовых контейнеров (disabled / normal), предотвращающее случайное редактирование результата пользователем.

    *Fluid Layout: Использование адаптивной сетки менеджера геометрии Grid. Элементы интерфейса масштабируются пропорционально изменению размеров главного окна.

## ⚙️ Требования и установка

    * Склонируйте репозиторий:
    ```bash

    git clone https://github.com/your-username/Modern-Translator-CustomTkinter.git
    cd Modern-Translator-CustomTkinter
```
    * Установите необходимые пакеты:
    ```bash

    pip install customtkinter deep-translator
```
    * Запустите скрипт:
    ```bash

    python main.py
```
## 🛠 Технологический стек

    * GUI: customtkinter (обертка над классическим tkinter с аппаратным сглаживанием скруглений и кастомными шейдерами фреймов).

    * Engine: deep-translator (GoogleTranslator модуль) — выбран как стабильный преемник устаревшей библиотеки googletrans.
